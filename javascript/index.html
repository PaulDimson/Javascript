Le basi di Javascript

Gli algoritmi sono una serie di passi da compiere per poter risolvere il problema in questione.

Le variabili sono contenitori che usiamo per poterci stabilire tutti progressi che abbiamo prodotto.

La differenza principale tra undefined e null è che undefined è usato per indicare un elemento inizialmente vuoto mentre null viene utilizzato per svuotare un elemento precedentemente pieno.
